Política de Privacidade

Privacy Policy

Kudos Manager

1. Informações Coletadas

1. Information Collected

A extensão Kudos Manager coleta apenas as seguintes informações:

The Kudos Manager extension collects only the following information:

  • Nomes dos membros da equipe (apenas primeiro nome)
  • Team member names (first name only)
  • Textos selecionados para criar kudos
  • Selected texts to create kudos
  • URLs e títulos das páginas onde os kudos foram criados
  • URLs and page titles where kudos were created
  • Mensagens e categorias dos kudos
  • Kudo messages and categories

2. Armazenamento dos Dados

2. Data Storage

Todos os dados são armazenados localmente no seu navegador utilizando a API chrome.storage.local. Nenhum dado é enviado para servidores externos.

All data is stored locally in your browser using the chrome.storage.local API. No data is sent to external servers.

3. Compartilhamento de Dados

3. Data Sharing

Não compartilhamos, vendemos ou enviamos seus dados para terceiros. Os dados permanecem exclusivamente no seu dispositivo.

We do not share, sell, or transmit your data to third parties. Data remains exclusively on your device.

4. Segurança

4. Security

Os dados são protegidos pelo navegador Chrome e ficam acessíveis apenas para esta extensão. Você pode apagar seus dados a qualquer momento através da função "Limpar Kudos".

Data is protected by the Chrome browser and is accessible only to this extension. You can delete your data at any time using the "Clear Kudos" function.

5. Seus Direitos

5. Your Rights

Você tem total controle sobre seus dados:

You have full control over your data:

  • Exportar todos os dados em formato JSON
  • Export all data in JSON format
  • Importar dados de outros dispositivos
  • Import data from other devices
  • Apagar todos os dados a qualquer momento
  • Delete all data at any time
  • Remover membros individualmente
  • Remove members individually
